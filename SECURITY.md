# Безопасность публичной копии

Репозиторий содержит демонстрационный исходный код и не должен содержать
рабочие пароли, токены, базы данных или персональные данные.

## Перед запуском

Создайте `.env` из `.env.example` и обязательно замените:

- `SECRET_KEY`;
- пароль пользователя MySQL;
- пароль администратора MySQL;
- разрешённые хосты и доверенные источники CSRF.

Демонстрационные учётные данные из команды `seed_demo` предназначены только
для локальной тестовой среды.

## Сообщение о проблеме

Не публикуйте обнаруженные пароли, персональные данные или подробности
уязвимости в открытом Issue. Свяжитесь с владельцем репозитория через профиль
GitHub: <https://github.com/itssyoma>.
