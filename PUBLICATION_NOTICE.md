# Сведения о публичной копии

## Автор

**Семён Баратов**

## Назначение

Этот репозиторий является публичной демонстрационной копией программного
продукта, разработанного в рамках выпускной квалификационной работы по теме:

> «Разработка электронного журнала в информационной системе колледжа».

Опубликованная версия предназначена для ознакомления проверяющих с исходным
кодом, архитектурой, тестами и документацией проекта.

## Состав копии

В репозитории находятся:

- исходный код веб-приложения;
- миграции структуры базы данных;
- шаблоны и статические файлы интерфейса;
- конфигурация Docker;
- автоматизированные тесты;
- руководство пользователя и техническая документация.

В репозиторий намеренно не включены:

- рабочая база данных;
- реальные сведения о студентах и сотрудниках;
- пароли, токены и серверные ключи;
- резервные копии;
- локальные файлы окружения `.env`;
- пояснительная записка с персональными и организационными сведениями;
- полная история закрытого рабочего репозитория.

## Версия

Публичный снимок подготовлен на основе версии **1.2.0**. Публичная история
Git начинается с отдельного контрольного коммита и не воспроизводит историю
основного рабочего репозитория.
