# История версий

## 1.2.0 — 08.06.2026

- Добавлено операционное логирование Django и Gunicorn.
- Добавлена ротация Docker-логов и инструкция для администратора.
- Добавлено логирование импорта, отчётов, занятий и архивирования.

## 1.1.1 — 08.06.2026

- Django обновлён с 5.2.3 до 5.2.15 для устранения известных уязвимостей.

## 1.1.0 — 08.06.2026

- Добавлены UI- и кросс-браузерные тесты Playwright.
- Добавлен нагрузочный сценарий Locust.
- Добавлены проверки безопасности Django, Bandit и `pip-audit`.
- Добавлена автоматическая проверка качества в GitHub Actions.
- Добавлены производственные настройки HTTPS и инструкция по тестированию.

## 1.0.0 — 08.06.2026

- Первый стабильный выпуск электронного журнала колледжа.
